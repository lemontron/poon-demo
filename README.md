## Poon Demo

Meteor app for demoing local Poon packages.

Run with:

```sh
npm run dev
```

Local Meteor packages are symlinked under `packages/`:

- `poon` -> `/Users/james/code/packages/poon`
- `poon-router` -> `/Users/james/code/packages/poon-router`
- `poon-markdown` -> `/Users/james/code/packages/poon-markdown`
- `poon-map` -> `/Users/james/code/packages/poon-map`
